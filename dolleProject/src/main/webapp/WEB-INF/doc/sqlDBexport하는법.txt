ⓓSQL Developer를 이용하여 데이터를 내보내는 방법

   1. SQL Developer -> 도구 -> 데이터베이스 익스포트
   2. [접속]에서 익스포트 할 계정 선택
   3. [DDL 익스포트]에서 종속항목을 제외한 모든 체크박스 체크
   4. 데이터가 많을 시 [다음마다 커밋 포함] 체크
   5. [다른 이름으로 저장]에서 <단일 파일>은 하나의 파일에 SQL문 전부 넣는 것이고,
      <별도의 디렉토리>는 각각 기능 별로 디렉토리에 나누어서 보관하는 것(별도의 디렉토리 추천)
   6. [인코딩]과 [파일]의 경로도 알맞게 설정
   7. 다음 -> 다음
   8. [조회] 클릭
   9. 익스포트 할 데이터들을 선택 (대부분 전부다 선택해야 함)
   10. 다음 -> 다음 -> 완료
